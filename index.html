COVID-19 Face Mask and Temperature Detection System

Project Overview:
This project aims to detect face masks and measure the temperature of individuals in real-time, providing an automated solution for ensuring safety in public spaces during the COVID-19 pandemic. The system enhances the detection of mask-wearing compliance and can be integrated with other security systems.

Technologies Used:

  • Python
  • OpenCV
  • TensorFlow
  • Keras
  • Deep Learning

Features:

  • Real-time face mask detection using convolutional neural networks (CNNs).
  • Automated temperature scanning using thermal cameras.
  • Integrated alert system for non-compliance and high temperature readings.
  • Scalable for deployment in various public spaces (airports, offices, malls).
